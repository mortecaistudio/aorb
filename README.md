# AORB website

Production-ready React + Vite website for Alliance of Rebels Europe.

## Preview locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

The deployable site is generated in `dist/`.

## Content and assets

- Website copy and sections: `src/App.jsx`
- Visual system and responsive layout: `src/styles.css`
- Production imagery: `public/assets/`
- Approved design concepts: `design/concepts/`
- Founding document download: `public/AORB_Brand_Identity_and_Manifesto.pdf`
